# Calculator

A calculator application built with HTML, CSS, and JavaScript as part of **The Odin Project Foundations** curriculum.

## Features

- Perform basic arithmetic operations
- Support for chained calculations
- Decimal number input
- Backspace functionality
- Keyboard support
- Division by zero protection
- Rounded decimal results
- Clear button to reset calculator state

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
